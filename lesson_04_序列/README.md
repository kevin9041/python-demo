# 第四章 序列（视频58-76）

## 序列（sequence）
    - 序列的分类：
        可变序列（序列中的元素可以改变）：
            > 列表（list）
        不可变序列（序列中的元素不能改变）：
            > 字符串（str）    
            > 元组（tuple）
## 字典（dict）
    - 在查询元素时，字典的效率是非常快的
## 集合（set）
    - 集合和列表非常相似
    - 不同点：
        1.集合中只能存储不可变对象
        2.集合中存储的对象是无序（不是按照元素的插入顺序保存）
        3.集合中不能出现重复的元素
