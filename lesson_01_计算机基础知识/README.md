## Sublime Text 3
    - 纯文本编辑器
    - Package Control（Sublime的包管理器）
        - Sublime Text3中的插件，通过该插件可以向Sublime中安装新的包
    - 练习：
        1.安装Sublime Text3
        2.在Sublime中安装Package Control
        3.通过Package Control安装Sublime的中文语言包   
        4.尝试为你的Sublime安装新的主题（通过搜索引擎搜索） ayu
